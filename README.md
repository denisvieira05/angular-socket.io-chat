# CHAT ANGULARJS + SOCKET.IO

### Implementação de um chat simples usando angularjs e socket.io .

#### 1. Install dependencies
```
npm install
``` 
```
bower install 
```
#### 1. Start MongoDB
```
mongod
```

#### 2. Start app.js 
```
node app.js
```
#### 3. APP IS RUNNING IN 
```
localhost:3000
```
